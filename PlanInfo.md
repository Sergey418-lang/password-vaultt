Проект: Консольный менеджер паролей (Password Vault)
План реализации (для студента)
Шаг 1. Структуры и хранение в памяти
•	Создать struct VaultEntry с полями: Service, Login, Password
•	Хранить все записи в []VaultEntry
•	Написать функции AddEntry(), ListEntries()
Шаг 2. Консольное меню
•	Бесконечный цикл с switch
•	Пункты: добавить, показать все, найти по сервису, удалить, выход
Шаг 3. Работа с файлом (JSON)
•	При запуске загружать vault.json
•	При каждом изменении — перезаписывать файл
•	Использовать json.MarshalIndent для читаемого формата
Шаг 4. Поиск и удаление
•	Поиск по части названия сервиса (strings.Contains)
•	Удаление по индексу (срез + append)
Шаг 5. Базовое шифрование (НЕ для реального использования, для портфолио)
•	Использовать crypto/aes или простой XOR с ключом
•	При сохранении шифровать пароль, при чтении расшифровывать
•	Ключ хранить в переменной окружения (os.Getenv)
Шаг 6. Мастер-пароль
•	При запуске спрашивать мастер-пароль
•	Если не совпадает с хешем (сохранённым в отдельном файле) — выход
•	Хеш через sha256
Шаг 7. Генератор паролей
•	Команда "сгенерировать пароль" (длина 12, цифры + буквы + символы)
•	Пакет crypto/rand
Шаг 8. Экспорт/импорт в CSV
•	Экспорт всех записей в vault.csv (без шифрования, только для резервной копии)
•	Импорт из CSV
