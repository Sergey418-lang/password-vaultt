# 🔐 Password Vault

[![Go Version](https://img.shields.io/badge/Go-1.21+-blue.svg)](https://go.dev)

Консольный менеджер паролей на Go с шифрованием.

## 🚀 Возможности

- 🔒 Мастер-пароль
- 🔑 XOR шифрование
- ➕ Добавление записей
- 🔍 Поиск по сервису
- 🎲 Генератор паролей
- 📤 Экспорт в CSV
