# 🔐 Password Vault

[![Go Version](https://img.shields.io/badge/Go-1.21+-blue.svg)](https://go.dev)

Консольный менеджер паролей на Go с шифрованием и резервным копированием.

## 🚀 Возможности

- 🔒 **Мастер-пароль** — защита доступа к хранилищу
- 🔑 **XOR шифрование** — пароли хранятся в зашифрованном виде
- ➕ **Добавление записей** — сервис, логин, пароль
- 📋 **Просмотр всех записей** — с расшифровкой на лету
- 🔍 **Поиск по сервису** — поиск по части названия
- ❌ **Удаление записей** — с подтверждением
- 🎲 **Генератор паролей** — криптобезопасная генерация
- 📤 **Экспорт в CSV** — резервная копия
- 📥 **Импорт из CSV** — восстановление данных
- 💾 **Сохранение в JSON** — автоматическое сохранение

## ⚙️ Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/ТВОЙ_ЛОГИН/password-vault.git

# Запустить
